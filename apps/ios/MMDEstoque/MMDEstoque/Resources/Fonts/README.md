# Fontes — MMD Estoque

Baixe os TTFs e coloque neste diretorio. Ja estao registradas no Info.plist (UIAppFonts).

## Space Grotesk
https://fonts.google.com/specimen/Space+Grotesk
Arquivos: SpaceGrotesk-Regular.ttf, SpaceGrotesk-Medium.ttf, SpaceGrotesk-SemiBold.ttf, SpaceGrotesk-Bold.ttf

## Space Mono
https://fonts.google.com/specimen/Space+Mono
Arquivos: SpaceMono-Regular.ttf, SpaceMono-Bold.ttf

## Doto
https://fonts.google.com/specimen/Doto
Arquivos: Doto-Regular.ttf, Doto-Bold.ttf

## Como baixar rapido

1. Acesse cada link acima no Google Fonts
2. Clique "Download family" (canto superior direito)
3. Extraia o ZIP e copie os TTFs para este diretorio
4. No Xcode, confirme que os arquivos aparecem no target MMDEstoque (Build Phases > Copy Bundle Resources)
